# ゼロからはじめるデータベース操作

書籍 [SQL 第2版 ゼロからはじめるデータベース操作](https://www.shoeisha.co.jp/book/detail/9784798144450) の学習用リポジトリ

以下の環境で実施

* Windows 10 pro Version 21H2
* Docker Desktop for Windows
  * Version: `4.8.1`
* Postgres Docker Official Image
  * `postgres:14.2`
* `Visual Studio Code`
  * 拡張機能: `Remote - Container`
